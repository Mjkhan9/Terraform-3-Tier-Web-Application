Modular Design

7 Terraform Modules

Clean, reusable infrastructure components following Terraform best practices.

🌐

VPC Module

Complete network infrastructure with public/private subnets, NAT Gateways, and VPC Flow Logs.

aws_vpc aws_subnet aws_nat_gateway aws_flow_log
🔒

Security Groups

Network isolation with least-privilege access rules for ALB, application, and database tiers.

aws_security_group ingress_rules egress_rules
⚖️

ALB Module

Application Load Balancer with target groups, health checks, and optional HTTPS support.

aws_lb aws_lb_target_group aws_lb_listener
📈

ASG Module

Auto Scaling Group with launch templates, user data scripts, and scaling policies.

aws_autoscaling_group aws_launch_template aws_autoscaling_policy
🗄️

RDS Module

PostgreSQL database with encryption, automated backups, and CloudWatch log exports.

aws_db_instance aws_db_subnet_group aws_db_parameter_group
📦

S3 Module

Encrypted S3 buckets for assets and ALB logs with versioning and lifecycle policies.

aws_s3_bucket aws_s3_bucket_versioning aws_s3_bucket_lifecycle
📊

CloudWatch Module

Monitoring and alerting with CPU alarms, response time tracking, and SNS notifications.

aws_cloudwatch_metric_alarm aws_sns_topic
Sample Code

Clean Terraform Code

Well-structured, documented infrastructure code following best practices.

📄 modules/vpc/main.tf 
# VPC with DNS support
resource "aws_vpc" "main" {
  cidr_block           = var.vpc_cidr
  enable_dns_hostnames = true
  enable_dns_support   = true

  tags = {
    Name = "${var.project_name}-vpc"
  }
}

# Public Subnets (Multi-AZ)
resource "aws_subnet" "public" {
  count = length(var.availability_zones)

  vpc_id                  = aws_vpc.main.id
  cidr_block              = cidrsubnet(var.vpc_cidr, 8, count.index)
  availability_zone       = var.availability_zones[count.index]
  map_public_ip_on_launch = true
}
📄 modules/asg/main.tf 
# Auto Scaling Group
resource "aws_autoscaling_group" "app" {
  name                = "${var.project_name}-asg"
  vpc_zone_identifier = var.private_subnet_ids
  target_group_arns   = [var.target_group_arn]
  health_check_type   = "ELB"

  min_size         = var.min_size
  max_size         = var.max_size
  desired_capacity = var.desired_capacity

  launch_template {
    id      = aws_launch_template.app.id
    version = "$Latest"
  }

  instance_refresh {
    strategy = "Rolling"
  }
}
📄 modules/security-groups/main.tf 
# RDS Security Group (Database Tier)
resource "aws_security_group" "rds" {
  name        = "${var.project_name}-rds-sg"
  description = "Security group for RDS database"
  vpc_id      = var.vpc_id

  ingress {
    description     = "PostgreSQL from app instances"
    from_port       = 5432
    to_port         = 5432
    protocol        = "tcp"
    security_groups = [aws_security_group.app.id]
  }
}
📄 modules/rds/main.tf 
# RDS PostgreSQL Instance
resource "aws_db_instance" "main" {
  identifier         = "${var.project_name}-db"
  engine             = "postgres"
  engine_version     = "15"
  instance_class     = var.db_instance_class
  allocated_storage  = var.db_allocated_storage
  storage_encrypted  = true

  db_name  = var.db_name
  username = var.db_username
  password = var.db_password

  backup_retention_period = 7
  skip_final_snapshot     = true
  publicly_accessible     = false
}
